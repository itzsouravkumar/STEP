# STEP - Student Training and Evaluation Program

Welcome to the **STEP (Student Training and Evaluation Program)** repository. This repository contains projects, exercises, and materials covered during various semesters of the program.

## Repository Structure

The repository is organized by semesters. Each semester directory contains its own specific projects, lab exercises, and session materials.

- **[Semester 2 (SEM-2)](./SEM-2)**: Covers core programming foundations, including Java methods, strings, arrays, and control flow.
- **[Semester 3 (SEM-3)](./SEM-3)**: Focuses on advanced Java programming fundamentals and comprehensive lab exercises.
- **[Semester 4 (SEM-4)](./SEM-4)**: Features practical application development and independent Java projects.

## centralized .gitignore

This repository uses a centralized `.gitignore` file at the root level to manage standard exclusions for Java development, various IDEs (IntelliJ, Eclipse, VS Code), and operating system-specific files.

## Getting Started

To explore the contents of a specific semester, navigate to the relevant directory and review its specific `README.md` file for detailed instructions on compilation, execution, and prerequisites.
