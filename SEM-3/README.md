# STEP - Java Learning Program

Basic STEP classes for learning Java programming fundamentals.

## Folder Structure

```
SEM-3/
└── WEEK-1/
    ├── LAB/          # Lab exercises and assignments
    ├── PRACTICE/     # Practice problems
    └── S-1/          # Session 1 code examples
```

## Description

- **LAB**: Laboratory exercises and assignments
- **PRACTICE**: Practice coding problems 
- **S-1**: Session materials and class examples

## Program Execution

```bash
# Compile Java file
javac FileName.java

# Run compiled program
java FileName
```

## Prerequisites

- Any Java IDE or text editor
