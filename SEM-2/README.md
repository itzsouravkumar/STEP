# STEP - Semester 2

This directory contains the materials and exercises for Semester 2 of the STEP program. The focus of this semester is on foundational Java programming concepts.

## Topics Covered

- **Java Methods**: Exercises and concepts related to defining and using methods in Java.
- **Java Strings**: Working with string manipulation and operations.
- **Programming Constructs**: Core building blocks of Java programming.
- **Programming Control Flows**: Understanding conditional statements (`if`, `switch`) and loops (`for`, `while`).
- **Programming with Arrays**: Processing and manipulating data structures using arrays.

## Running Java Programs

To run the Java files found in these directories:

1. Open your terminal or command prompt.
2. Navigate to the directory containing the `.java` file.
3. Compile the file:
   ```bash
   javac FileName.java
   ```
4. Run the compiled class:
   ```bash
   java FileName
   ```
